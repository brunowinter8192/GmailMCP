# Auth

**Scope:** `gmail.readonly` — read and search only, no write access.

**Credentials location:** `~/.gmail-mcp/`
- `gcp-oauth.keys.json` — OAuth client config (from Google Cloud Console)
- `credentials.json` — User token (generated by `auth/setup.py`)

**Setup:** `python3 auth/setup.py` — one-time browser-based OAuth flow.

**Runtime:** `src/gmail/auth.py` loads credentials, auto-refreshes expired tokens.
