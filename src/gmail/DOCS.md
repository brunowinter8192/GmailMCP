# src/gmail/ - Gmail API Modules

## Directory Structure

```
src/gmail/
  __init__.py
  auth.py                # OAuth credential loading + auto-refresh
  search_emails.py       # Search orchestration
  list_emails.py         # List by label orchestration
  read_email.py          # Full message read orchestration
  formatting.py          # Email formatting (summary + full views)
```

---

## auth.py

**Purpose:** Load OAuth credentials from `~/.gmail-mcp/`, auto-refresh expired tokens.

**Input:** None (reads from filesystem)
**Output:** Authenticated Gmail API service or Credentials object

**Usage:**
```python
from src.gmail.auth import get_gmail_service, load_credentials

service = get_gmail_service()  # Returns authenticated Gmail API service
creds = load_credentials()     # Returns Credentials object (for direct API calls)
```

**Constants:**
| Constant | Value | Description |
|----------|-------|-------------|
| SCOPES | `["...gmail.readonly"]` | OAuth scope (read-only) |
| CONFIG_DIR | `~/.gmail-mcp/` | Credential storage directory |
| CREDENTIALS_PATH | `~/.gmail-mcp/credentials.json` | User token file |

**Functions:**
| Function | Purpose |
|----------|---------|
| `get_gmail_service()` | Build authenticated Gmail API service via google-api-python-client |
| `load_credentials()` | Load credentials from disk, refresh if expired |

**Credential files:**
| File | Source | Description |
|------|--------|-------------|
| `gcp-oauth.keys.json` | Google Cloud Console download | OAuth client config |
| `credentials.json` | Generated by `auth/setup.py` | User token (auto-refreshed) |

## search_emails.py

**Purpose:** Search Gmail by query string using Gmail's search syntax.

**Input:** Query string, max results
**Output:** `list[TextContent]` with formatted email summaries

**Usage:**
```python
from src.gmail.search_emails import search_emails_workflow

results = search_emails_workflow("from:boss@company.com subject:report", max_results=5)
```

**Functions:**
| Function | Purpose |
|----------|---------|
| `search_emails_workflow(query, max_results)` | Orchestrator: search, fetch metadata, format summaries |
| `fetch_message_list(service, query, max_results)` | Call messages.list API with search query |
| `fetch_message_summaries(service, messages)` | Fetch metadata (Subject, From, To, Date) per message |

| Function | Purpose |
|----------|---------|

| `fetch_full_message(service, message_id)` | Call messages.get API with format=full |

## formatting.py

**Purpose:** Format Gmail API message objects into human-readable text output.

**Input:** Gmail message dicts (from API)
**Output:** Formatted strings

**Usage:**
```python
from src.gmail.formatting import format_email_summary, format_email_full

summary = format_email_summary(msg)  # Compact: ID, date, from, subject, snippet
full = format_email_full(msg)        # Complete: headers, body, attachments
```

**Functions:**
| Function | Purpose |
|----------|---------|
| `get_header(headers, name)` | Extract header value by name (case-insensitive) |
| `format_date(internal_date_ms)` | Convert Gmail internalDate (ms epoch) to `YYYY-MM-DD HH:MM` |
| `extract_body(payload)` | Extract plain text from MIME payload (handles multipart, nested) |
| `extract_attachments(payload)` | Extract attachment filenames with sizes |
| `strip_html(html)` | Convert HTML to plain text (br → newline, remove tags/scripts) |
| `clean_body(text)` | Remove tracking URLs, footers, zero-width chars, collapse whitespace |
| `format_email_summary(msg)` | Format metadata view: ID, date, from, subject, snippet |
| `format_email_full(msg)` | Format full view: all headers, cleaned body, attachment list |

**Body extraction priority:**
1. `text/plain` (preferred)
2. `text/html` → stripped via `strip_html()` + `clean_body()`
3. Nested multipart → recursive extraction

## auth/setup.py

**Purpose:** One-time OAuth consent flow. Opens browser, saves credentials.

**Usage:**
```bash
python3 auth/setup.py
```

**Prerequisites:**
1. Google Cloud project with Gmail API enabled
2. OAuth Desktop client JSON at `~/.gmail-mcp/gcp-oauth.keys.json`

**Output:** Saves token to `~/.gmail-mcp/credentials.json`
